Como usar este app?

  1. Clique em "Sortear Tema" para escolher um assunto aleatório
  2. Clique em "Sortear Tempo Verbal" para selecionar um tempo verbal em inglês
  3. Use "Mostrar Estrutura" para ver como formar frases neste tempo verbal
  4. Clique em "Mostrar Exemplo" para ver um exemplo contextualizado com o tema

Dica: Tente criar suas próprias frases usando a estrutura mostrada!
